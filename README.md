## Getting Started

To begin using the API, follow these steps:

1. **Obtain API Keys**: Sign up for an account and obtain your API keys from the developer portal.
2. **Set Up Your Environment**: Configure your development environment to include the necessary libraries for making HTTP requests.
3. **Make Your First Request**: Use the provided endpoints to start converting SSE streams to Poe Protocol format or vice versa.
